Google Apps Scriptで始める業務自動化の基礎

2025年5月15日
Google Apps Script
読了時間: 約10分

多くの企業では、日々の業務の中で繰り返し行われる単調な作業に多くの時間が費やされています。データ入力、レポート作成、情報の集約など、これらの作業は重要でありながらも、本来注力すべき創造的な業務から時間を奪っています。Google Apps Script（GAS）は、そんな課題を解決するための強力なツールです。

Google Apps Scriptとは

Google Apps Script（GAS）は、Google Workspaceのアプリケーション（スプレッドシート、ドキュメント、フォームなど）を自動化・拡張するためのJavaScriptベースのスクリプト言語です。GASを使うことで、プログラミングの知識が少なくても、業務プロセスの自動化が可能になります。

GASの主な特徴は以下の通りです：

  • Google Workspaceとの完全な互換性
  • クラウド上で動作するため、サーバー不要
  • JavaScriptベースで学習障壁が低い
  • 無料で利用可能（一定の制限内）
  • 豊富なAPIとライブラリ

GASで何ができるのか

GASを活用することで、以下のような業務自動化が可能になります：

1. データの自動集計・分析

複数のスプレッドシートやフォームからデータを自動的に収集し、集計・分析するスクリプトを作成できます。例えば、日々の売上データを自動的に集計し、週次・月次レポートを生成することが可能です。

2. 定期レポートの自動生成

定期的に作成する必要のあるレポートを自動化できます。データの取得から、グラフの作成、ドキュメントへの挿入、関係者へのメール送信まで、一連のプロセスを自動化することが可能です。

3. フォーム連携の自動処理

Googleフォームの回答を自動的に処理し、適切なアクションを実行するスクリプトを作成できます。例えば、問い合わせフォームの回答に応じて、自動返信メールを送信したり、担当者に通知したりすることが可能です。

4. カスタムUI開発

スプレッドシートやドキュメントに、カスタムメニューやサイドバー、ダイアログなどのUIを追加できます。これにより、非技術者でも簡単に操作できるインターフェースを提供することが可能です。

初めてのGoogle Apps Scriptプログラム

それでは、実際にGASを使った簡単なプログラムを作成してみましょう。ここでは、スプレッドシートのデータを自動的に整形し、サマリーシートを作成する例を紹介します。

ステップ1: スクリプトエディタを開く

Google スプレッドシートを開き、「拡張機能」→「Apps Script」をクリックします。これにより、スクリプトエディタが開きます。

ステップ2: コードを記述する

以下のコードをエディタに貼り付けます。このスクリプトは、「データ」シートの情報を集計し、「サマリー」シートに結果を表示します。

function createSummary() {
  // スプレッドシートを取得
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  
  // データシートを取得
  var dataSheet = ss.getSheetByName('データ');
  if (!dataSheet) {
    Browser.msgBox('「データ」シートが見つかりません。');
    return;
  }
  
  // データ範囲を取得
  var dataRange = dataSheet.getDataRange();
  var values = dataRange.getValues();
  
  // ヘッダー行を取得
  var headers = values[0];
  
  // 金額列のインデックスを取得
  var amountIndex = headers.indexOf('金額');
  if (amountIndex === -1) {
    Browser.msgBox('「金額」列が見つかりません。');
    return;
  }
  
  // カテゴリ列のインデックスを取得
  var categoryIndex = headers.indexOf('カテゴリ');
  if (categoryIndex === -1) {
    Browser.msgBox('「カテゴリ」列が見つかりません。');
    return;
  }
  
  // カテゴリごとの合計を計算
  var categorySums = {};
  for (var i = 1; i < values.length; i++) {
    var category = values[i][categoryIndex];
    var amount = values[i][amountIndex];
    
    if (!categorySums[category]) {
      categorySums[category] = 0;
    }
    
    categorySums[category] += amount;
  }
  
  // サマリーシートを作成または取得
  var summarySheet = ss.getSheetByName('サマリー');
  if (!summarySheet) {
    summarySheet = ss.insertSheet('サマリー');
  } else {
    summarySheet.clear();
  }
  
  // サマリーデータを作成
  var summaryData = [['カテゴリ', '合計金額']];
  for (var category in categorySums) {
    summaryData.push([category, categorySums[category]]);
  }
  
  // サマリーシートにデータを書き込む
  summarySheet.getRange(1, 1, summaryData.length, 2).setValues(summaryData);
  
  // グラフを作成
  var chartRange = summarySheet.getRange(1, 1, summaryData.length, 2);
  var chart = summarySheet.newChart()
    .setChartType(Charts.ChartType.PIE)
    .addRange(chartRange)
    .setPosition(5, 5, 0, 0)
    .setOption('title', 'カテゴリ別支出')
    .setOption('pieSliceText', 'percentage')
    .build();
  
  summarySheet.insertChart(chart);
  
  Browser.msgBox('サマリーが作成されました！');
}

ステップ3: 実行する

スクリプトを保存し、「実行」ボタンをクリックします。初回実行時は、スクリプトの権限を承認する必要があります。承認後、スクリプトが実行され、サマリーシートが作成されます。

ステップ4: トリガーを設定する（オプション）

このスクリプトを定期的に実行したい場合は、トリガーを設定することができます。スクリプトエディタの「トリガー」メニューから、新しいトリガーを作成し、実行頻度を設定します。

業務自動化の効果

GASを活用した業務自動化の効果は、多くの企業で実証されています。以下は、実際の導入事例から得られた効果の一例です：

  • 月次レポート作成時間: 2日間 → 30分（約96%削減）
  • データ入力ミス: 月平均5件 → 0件（100%削減）
  • 情報共有の遅延: 平均24時間 → リアルタイム
  • 従業員の満足度: 25%向上

これらの効果は、単なる時間削減だけでなく、業務品質の向上、従業員のモチベーション向上、そして最終的には顧客満足度の向上にもつながっています。

まとめ

Google Apps Scriptは、業務自動化の入門として最適なツールです。プログラミングの経験が少なくても、比較的短時間で効果的な自動化ソリューションを構築することができます。

まずは小さな業務から自動化を始め、徐々に範囲を広げていくことをお勧めします。自動化によって生まれた時間を、より創造的で価値の高い業務に充てることで、個人としても組織としても大きな成長につながるでしょう。
